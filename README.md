## torpedo.mkb.ru

## Файловая структура

```
webpack-frontend-template
├── dist
├── public
|   ├── index.html
├── src
│   ├── assets
│       ├── fonts
│       └── images
│   ├── pages
│   ├── modules
│   ├── components
│   ├── ui
│   ├── styles
│   └── index.ts
├── webpack.config.js
├── package.json
├── .eslintignore
├── .eslintrc.json
├── .gitignore
└── .prettierrc
```


## Команды

* ```npm run dev``` - собираем development
* ```npm run build``` - собираем production
* ```npm run lint``` - запуск анализа линтера
* ```npm run lint:fix``` - запуск линтера в режиме --fix

## Установка

Установим все необходимые пакеты из package.json

```bash
npm install
```

Запускаем

```bash
npm start
```
